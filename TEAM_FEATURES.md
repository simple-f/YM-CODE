## 📋 配置清单

### YM-CODE（个人）

```bash
# 1. 安装
pip install -r requirements.txt

# 2. 运行
python -m ymcode

# 3. 使用
> 读取 src/main.py
> 执行 git status
> 运行测试
```

**时间：** 5 分钟  
**配置：** 无需配置

---

### OpenClaw（团队）

```bash
# 1. 安装
npm install -g openclaw

# 3. 配置渠道

# 4. 配置共享记忆
# 团队知识库
```

**时间：** 30 分钟  
**配置：** 需要配置

### 工作流程

```
1. Alice 开发功能
   └─> YM-CODE 本地编辑
   └─> Git 提交代码

2. Bob 开发 API
   └─> YM-CODE 本地编辑
   └─> Git 提交代码

3. Charlie 测试

   └─> OpenClaw 通知团队

4. 团队协作
   └─> OpenClaw 共享记忆
   └─> Feishu 沟通
```
