## 🚀 使用示例

### 启动 CLI

```bash
# 直接运行
python -m ymcode.cli.app

# 或使用入口点
ym-code
```

### 面板使用

```python
from ymcode.cli.panels import WelcomePanel, StatusPanel, InfoPanel

# 欢迎面板
welcome = WelcomePanel(version="1.0.0")
welcome.display()

# 状态面板
status = StatusPanel()
status.update("MCP Servers", "5")
status.update("Skills", "6")
status.display()

# 信息面板
info = InfoPanel()
info.add_success("初始化完成")
info.add_error("连接失败")
info.add_warning("配置缺失")
info.display()
```

### 主应用

```python
from ymcode.cli.app import YMCodeApp

# 创建应用
app = YMCodeApp()

# 运行
app.run_sync()
```
