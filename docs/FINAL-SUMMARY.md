## 🚀 使用指南

### 快速开始

```bash
# 1. 进入目录
cd ymcode

# 2. 启动 CLI
python cli.py

# 3. 输入任务
YM-CODE> 帮我设计一个微服务架构

# 4. 查看路由效果
🟢 智能路由 0.5ms
  Agent: 架构师 (ai1)
  置信度：🟢 75%
  预计响应：5s
  关键词：微服务，架构
  备选：技术顾问
```

### API 调用

```python
from ymcode.router import SmartRouter

# 创建路由器
router = SmartRouter()

# 路由任务
result = router.route("帮我设计微服务架构")

# 获取结果
print(f"Agent: {result.agent_name}")
print(f"置信度：{result.confidence:.0%}")
print(f"关键词：{result.matched_keywords}")

# 记录结果
router.record_result("ai1", True, 3.5)
```
