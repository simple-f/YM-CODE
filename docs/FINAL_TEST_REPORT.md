## 📋 使用方式

### 1. 访问主界面

```
http://localhost:18770
```

**新建项目方式：**
- 方式 1：点击右上角 **[+ ]** 按钮
- 方式 2：点击侧边栏 **[👥 Agent 配置]**
- 方式 3：直接访问 `http://localhost:18770/agent-config.html`

### 2. 技能管理

```
侧边栏 → [🛠️ 技能管理]
```

- 所有技能已内置，无需下载
- 可以启用/禁用技能
- 技能状态实时显示

### 3. Agent 配置

```
侧边栏 → [👥 Agent 配置]
```

- 选择需要的 Agent
- 创建工作区
- 保存配置
