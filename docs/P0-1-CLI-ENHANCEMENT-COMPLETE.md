### 4. 组件 API

#### TaskPanel - 任务面板
```python
from ymcode.cli.enhanced import TaskPanel

task = TaskPanel()
task.start("分析代码结构")
# ... 执行任务 ...
task.complete(success=True, message="代码分析完成")
```

#### EnhancedProgress - 进度条
```python
from ymcode.cli.enhanced import EnhancedProgress

progress = EnhancedProgress()
with progress.create("处理中...") as p:
    task = p.add_task("", total=100)
    for i in range(0, 101, 10):
        p.update(task, completed=i)
```

#### InfoBox - 信息框
```python
from ymcode.cli.enhanced import InfoBox

box = InfoBox()
box.success("标题", "消息")
box.info("标题", "消息")
box.warning("标题", "消息")
box.error("标题", "消息", "建议")
```

#### StatsTable - 统计表格
```python
from ymcode.cli.enhanced import StatsTable

table = StatsTable()
table.show("系统状态", {
    "CPU 使用率": "45%",
    "内存使用": "2.1GB / 8GB"
})
```

#### CommandHelp - 命令帮助
```python
from ymcode.cli.enhanced import CommandHelp

help = CommandHelp()
help.show({
    "help": "显示帮助信息",
    "exit": "退出程序"
})
```

## 📝 使用说明

### 使用美化版 App
```bash
# 运行美化版
python -m ymcode.cli.app_enhanced
```

### 使用组件库
```python
from ymcode.cli.enhanced import (
    TaskPanel, EnhancedProgress, InfoBox,
    StatsTable, CommandHelp
)

# 创建任务面板
task = TaskPanel()
task.start("任务名称")
# ... 执行 ...
task.complete(success=True)

# 显示进度
progress = EnhancedProgress()
with progress.create("处理中...") as p:
    # ... 更新进度 ...
    pass

# 显示信息
box = InfoBox()
box.success("成功", "操作完成")
```
