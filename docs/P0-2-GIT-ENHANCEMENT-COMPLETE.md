## API 使用

### 基础使用
```python
from ymcode.tools.git_tools_enhanced import EnhancedGitTool
from rich.console import Console

console = Console()
git = EnhancedGitTool(console)

# 可视化状态
git._status_visual()

# 可视化 diff
git._diff_visual(staged=False)

# 提交历史
await git.show_log_visual(limit=10)

# 分支管理
await git.branch_management()
```

### 交互式 Commit
```python
# 交互式提交（带 UI）
result = await git.interactive_commit()
```

### 集成到 YM-CODE
```python
# 在 YM-CODE 中使用
from ymcode.tools.git_tools_enhanced import create_enhanced_git_tool

git_tool = create_enhanced_git_tool(console)

# 注册为工具
agent.tools.register(git_tool)
```
