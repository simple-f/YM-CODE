## 🔄 核心工作流程

### 1. 用户请求处理流程

```
用户输入
   │
   ▼
┌─────────────────┐
│  Web UI / CLI   │
└─────────────────┘
   │
   ▼
┌─────────────────┐
│  API Gateway    │
└─────────────────┘
   │
   ▼
┌─────────────────┐
│  Core Agent     │
│  - 解析输入     │
│  - 加载上下文   │
│  - 规划执行     │
└─────────────────┘
   │
   ▼
┌─────────────────┐
│  LLM 调用        │
│  - 生成思考     │
│  - 选择工具     │
└─────────────────┘
   │
   ▼
┌─────────────────┐
│  Tool Execution │
│  - Skills       │
│  - MCP Tools    │
└─────────────────┘
   │
   ▼
┌─────────────────┐
│  Response       │
│  - 整合结果     │
│  - 更新记忆     │
│  - 返回用户     │
└─────────────────┘
```

### 2. 多 Agent 协作流程

```
用户任务
   │
   ▼
┌─────────────────┐
│  A2A Router     │
│  - 分析任务     │
│  - 路由分发     │
└─────────────────┘
   │
   ├──────────────┐
   ▼              ▼
┌─────────┐  ┌──────────┐
│ Builder │  │Reviewer  │
│ Agent   │  │Agent     │
│ - 实现  │  │ - 审查   │
│ - 构建  │  │ - 质量   │
└─────────┘  └──────────┘
   │              │
   └──────┬───────┘
          ▼
   ┌─────────────┐
   │ Handoff     │
   │ - 5 件套格式  │
   │ - 状态追踪  │
   └─────────────┘
```
