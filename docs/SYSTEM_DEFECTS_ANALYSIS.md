**目标:**

    all=True

### 6. 测试智能生成弱

**问题:** 只能生成基础测试，没有边界条件、Mock 数据

**影响:** 测试覆盖率低

**现状:**
```python
def test_add():
    assert add(1, 2) == 3
```

**目标:**
```python
def test_add():
    # 正常情况
    assert add(1, 2) == 3
    assert add(-1, -2) == -3
    
    # 边界条件
    assert add(0, 0) == 0
    assert add(MAX_INT, 0) == MAX_INT
    
    # 异常情况
    with pytest.raises(TypeError):
        add("1", "2")
```

**预计工时:** 3 天
