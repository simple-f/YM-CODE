```

### 用户数据目录

```
~/.ym-code/                 # 用户数据
├── config.yaml             # 个人配置
├── skills/                 # 已安装技能
├── memory/                 # 记忆存储

## 📁 配置文件示例

### 主配置 (`~/.ym-code/config.yaml`)

```yaml
# YM-CODE 配置文件
# 你的编程伙伴，你的思维延伸

app:
  name: "YM-CODE"
  version: "0.1.0"
  codename: "Aurora"

user:
  nickname: ""  # 你的称呼
  timezone: "Asia/Shanghai"
  language: "zh-CN"

ai:
  default_model: "qwen3.5-plus"
  max_tokens: 4000
  timeout: 30

ui:
  theme: "aurora"  # aurora/light/dark/matrix/cyberpunk
  show_progress: true
  syntax_highlight: true

memory:
  enabled: true
  auto_compress: true
  session_retention_days: 30
```

### 技能配置 (`~/.ym-code/skills.yaml`)

```yaml
# YM-CODE 技能配置

packages:
  core:
    enabled: true
    skills:
      - search
      - http
      - shell
      - code_analysis
  
  development:
    enabled: true
    skills:
      - database
      - formatter
      - docker

workflows:
  code-review:
    description: "自动代码审查流程"
    steps:
      - skill: code_analysis
      - skill: shell
        command: "git diff"
```

### 项目配置 (`.ymcoderc`)

```yaml
# YM-CODE 项目配置

project:
  name: "ym-code"
  type: "python"

skills:
  enabled:
    - code_analysis
    - shell
    - formatter

context:
  auto_index: true
  ignore:
    - node_modules/
    - __pycache__/
```
