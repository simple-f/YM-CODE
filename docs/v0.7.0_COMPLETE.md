## 📦 新增功能

### 1. 统一 LLM 客户端

**文件：**
- `ymcode/core/llm_client.py` - 统一客户端
- `ymcode/core/api_model.py` - API 模型

**功能：**
- ✅ 支持 API 和本地两种模式
- ✅ 配置驱动，可选启用
- ✅ 向后兼容现有代码

**使用示例：**
```python
# API 模式（默认）
client = LLMClient()

# 本地模式（可选）
client = LLMClient({'use_local': True})

# 接口不变
response = client.chat("你好")
```

---

### 2. 上下文管理器

**文件：**
- `ymcode/core/context_manager.py` - 上下文管理

**功能：**
- ✅ 自动检测长上下文
- ✅ 智能压缩（保留关键信息）
- ✅ 分片处理（超长上下文）
- ✅ 接口不变，透明处理

**使用示例：**
```python
from ymcode.core.context_manager import process_context

# 自动处理长上下文
processed = process_context(context, max_tokens=8000)
```

---

### 3. 代码分析技能

**文件：**
- `ymcode/skills/code_analyzer.py` - 代码分析技能
- `ymcode/skills/analyzers/pylint_analyzer.py` - Pylint 分析器
- `ymcode/skills/analyzers/black_formatter.py` - Black 格式化器
- `ymcode/skills/analyzers/flake8_checker.py` - Flake8 检查器

**功能：**
- ✅ 集成 Pylint 代码检查
- ✅ 集成 Black 代码格式化
- ✅ 集成 Flake8 代码检查
- ✅ 可选依赖，按需启用
- ✅ 独立技能，不影响现有

**使用示例：**
```python
from ymcode.skills.code_analyzer import CodeAnalyzerSkill

skill = CodeAnalyzerSkill()
result = await skill.execute({
    'code': 'def hello():\n    print("world")',
    'language': 'python'
})
```
