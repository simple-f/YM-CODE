**功能：**
- ✅ 支持 API 和本地两种模式
- ✅ 配置驱动，可选启用
- ✅ 向后兼容现有代码
- ✅ 自动回退机制

**使用示例：**
```python

client = LLMClient()

# 或使用本地模型（可选）

    'use_local': True,
    'local_model_path': '~/.ymcode/models/qwen-7b'
})

# 接口不变
response = client.chat("你好")
```

### 任务 2：上下文管理器

**新增文件：**
- `ymcode/core/context_manager.py` - 上下文管理

**功能：**
- ✅ 自动检测长上下文
- ✅ 智能压缩（保留关键信息）
- ✅ 分片处理（超长上下文）
- ✅ 接口不变，透明处理

**使用示例：**
```python

# 自动处理长上下文
```
