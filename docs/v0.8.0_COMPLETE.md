**文件：**

**功能：**
- ✅ 代码分析（Pylint、Flake8）
- ✅ 代码格式化（Black）
- ✅ 实时问题检测
- ✅ 保存时自动分析
- ✅ 右键菜单集成

**使用：**
```typescript

YM-CODE: 分析当前文件

```

**功能：**
- ✅ Git 仓库克隆

**使用：**
```python

})
```

## 📝 安装说明

### 基础安装（必需）

```bash

```

```bash

npm install
npm run compile
# 在 VSCode 中加载插件
```
