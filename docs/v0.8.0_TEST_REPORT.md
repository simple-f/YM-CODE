## 📝 安装指南

### 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/simple-f/YM-CODE.git
cd YM-CODE

# 2. 安装依赖
pip install -r requirements.txt

# 3. 初始化
python init.py

# 4. 启动
python start-web.py

# 5. 访问
# http://localhost:18770
```

### VSCode 插件

```bash
# 1. 进入插件目录
cd extensions/vscode

# 2. 安装依赖
npm install

# 3. 编译
npm run compile

# 4. 在 VSCode 中加载
# 开发者工具 → 加载未打包的扩展
```
