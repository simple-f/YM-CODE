## 📦 新增功能

### 1. 多语言支持 ✅

**文件：**
- `ymcode/skills/multi_language.py` - 多语言技能

**支持语言：**
- ✅ JavaScript/TypeScript
- ✅ Java
- ✅ Go
- ✅ Python（已有）

**功能：**
- ✅ 代码分析
- ✅ 语言检测
- ✅ 语法特性识别
- ✅ 复杂度评估

**使用示例：**
```python
skill = MultiLanguageSkill()
result = await skill.execute({
    'code': 'function hello() { console.log("world"); }',
    'language': 'javascript',
    'action': 'analyze'
})
```

### 2. 代码运行/调试 ✅

**文件：**
- `ymcode/skills/code_runner.py` - 代码运行技能

**功能：**
- ✅ 安全沙箱执行
- ✅ Python 代码运行
- ✅ JavaScript/Node 代码运行
- ✅ 输出捕获
- ✅ 错误处理
- ✅ 超时控制
- ✅ 环境变量隔离

**使用示例：**
```python
skill = CodeRunnerSkill()
result = await skill.execute({
    'code': 'print("Hello, World!")',
    'language': 'python',
    'timeout': 10
})
# 返回：stdout, stderr, exit_code
```
