**目标：** 支持主流编程语言

**支持语言：**
- [x] Python（已有）
- [ ] JavaScript/TypeScript
- [ ] Java
- [ ] Go
- [ ] C/C++

**技术方案：**
```python
# ymcode/skills/multi_language.py
class MultiLanguageSkill(BaseSkill):
    """多语言支持技能"""
    
    def __init__(self):
        self.languages = {
            'python': PythonHandler(),
            'javascript': JavaScriptHandler(),
            'typescript': TypeScriptHandler(),
            'java': JavaHandler(),
            'go': GoHandler()
        }
    
    async def analyze(self, code: str, language: str):
        handler = self.languages.get(language)
        if handler:
            return handler.analyze(code)
```

**预计工作量：** 3-4 天
