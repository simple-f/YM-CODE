## 📝 安装指南

### 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/simple-f/YM-CODE.git
cd YM-CODE

# 2. 安装依赖
pip install -r requirements.txt

# 3. 初始化
python init.py

# 4. 启动
python start-web.py

# 5. 访问
# http://localhost:18770
```

### 运行时支持（可选）

```bash
# Node.js（运行 JavaScript）
node --version
```
