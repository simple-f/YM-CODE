## 🔧 问题修复（v0.3.1）

### 修复的问题

1. **Skills 工具未注册到 Agent**
   - 问题：Skills 工具没有注册到 Agent 的工具注册表
   - 修复：在 `initialize_skills()` 中添加 `self.tools.register_mcp_tool(tool_def)`
   - 位置：`ymcode/core/agent.py`

2. **Skills 工具定义未传递给 LLM**
   - 问题：Agent.run() 只传递了本地工具，没有包含 Skills
   - 修复：在调用 LLM 前添加 Skills 工具定义
   - 位置：`ymcode/core/agent.py` 的 `run()` 方法

3. **ToolRegistry 缺少 MCP 工具注册方法**
   - 问题：没有 `register_mcp_tool()` 方法
   - 修复：添加该方法用于记录 MCP 工具
   - 位置：`ymcode/tools/registry.py`
