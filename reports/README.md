# 报告文件

此目录包含历史报告、文档和临时文件。

## 目录结构

- **历史报告** - 各版本的开发报告、测试报告
- **设计文档** - 架构设计、功能设计
- **配置指南** - 各种配置说明
- **临时文件** - 开发过程中的临时文档

## 核心文档

核心文档请查看根目录的 `docs/` 目录：

- README.md - 项目说明
- INSTALL.md - 安装指南
- CHANGELOG.md - 变更日志
- docs/ARCHITECTURE.md - 架构说明
- docs/USAGE.md - 使用指南
- docs/API.md - API 文档
