## 8. 快速开始指南

### 环境要求

- Python 3.10+
- 内存 2GB+
- 磁盘 1GB+

### 安装步骤

```bash
# 1. 进入项目目录
cd shared/YM-CODE

# 2. 创建虚拟环境 (可选)
python -m venv venv
venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置 API Key (已配置)
# .env 文件已包含 DASHSCOPE_API_KEY

# 5. 测试导入
python -c "from ymcode.core.agent import Agent; print('OK')"

# 6. 启动 Web 界面 (可选)
python start-web.py

# 7. 访问界面
# http://localhost:18770
```

### 使用示例

```python
# 使用 Agent
from ymcode.core.agent import Agent

agent = Agent(config={
    'max_iterations': 5,
    'timeout': 60
})

result = await agent.run("帮我创建一个 Python 项目")
print(result)

# 使用技能
from ymcode.skills.shell import ShellSkill

shell = ShellSkill()
result = await shell.execute({
    'command': 'git status'
})
print(result)

# 使用工具
from ymcode.tools.file_tools import ReadFileTool

tool = ReadFileTool()
content = await tool.execute({
    'path': 'README.md'
})
print(content)
```
