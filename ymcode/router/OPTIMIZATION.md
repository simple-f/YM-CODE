- 这个用例处于边界情况

1. 方案 A: 将"技术选型"规则权重提高（已做）

**决定：** 接受为边界情况，不强制修复

### 3. 置信度计算优化

```python
def calculate_confidence(self, score, matched_keywords):
    # 基础置信度
    base = min(0.7, 0.3 + score * 0.1)
    
    # 多样性奖励
    diversity = min(0.2, len(matched_keywords) * 0.05)
    
    # 高优先级奖励
    if self.is_high_priority(matched_keywords):
        base += 0.2
    
    return min(1.0, base + diversity)
```
